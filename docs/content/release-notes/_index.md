What shipped in each release, newest first. Every tagged version builds the
same set of artifacts: archives for Linux, macOS, and Windows on amd64 and
arm64, Linux packages (deb, rpm, apk), a multi-arch container image on GHCR,
and entries for the package managers. Binaries are pure Go, so there is
nothing to install alongside them.
