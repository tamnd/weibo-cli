What shipped in each release, newest first. Every tagged version builds the
same set of artifacts: archives for Linux (amd64, arm64, armv7, 386), macOS
(amd64, arm64), Windows (amd64, arm64), and FreeBSD (amd64, arm64), Linux
packages (deb, rpm, apk), a multi-arch container image on GHCR, and entries
for the package managers. Binaries are pure Go, so there is nothing to install
alongside them.
